# React ☙ D3 — Barplot

A practice project I built for the [D3 ❤️ React course](https://www.react-graph-gallery.com/react-d3-dataviz-course).

## What's in here

A horizontal bar chart showing the number of students per country, featuring:

- Band scale (Y) + linear scale (X) via D3
- Custom `AxisLeft` and `AxisBottom` components built from scratch in JSX
- Hover interactions using `useState`
- Animated tooltip with a whimsical curvy line
- Styled with DM Mono + Caveat fonts

## Stack

- React
- D3
- Vite
